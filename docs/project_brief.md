# Project

## Drivers of Food Insecurity in London, United Kingdom

---

# Project Description

This project investigates the drivers of food insecurity and energy insecurity among London residents using a real-world survey dataset of approximately 3,000 participants.

The dataset contains demographic information, deprivation indicators and anonymised geographical information that can be linked with environmental factors such as crime rates and access to green space.

The project will involve:

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering
- Machine learning classification
- Machine learning model development
- Model evaluation and interpretation
- Development of an interactive web application for prediction

The web application serves as a practical demonstration of the trained models, allowing users to estimate their food insecurity and energy insecurity risk based on demographic, household, and area-level information.

---

# Project Aims

- Produce a cleaned and prepared dataset suitable for analysis.
- Perform exploratory data analysis to understand the characteristics and quality of the dataset.
- Engineer additional features, including environmental variables such as crime rate and access to green space.
- Develop and compare machine learning classification models for predicting food insecurity and energy insecurity.
- Evaluate the contribution of individual features to the model predictions using interpretable machine learning techniques.
- Analyse and discuss the underlying drivers of food insecurity and energy insecurity based on the model outcomes.
- Develop a web application to demonstrate the trained prediction models.

---

# Project Deliverables

The project will deliver:

- A cleaned and documented dataset.
- An exploratory data analysis report supported by appropriate visualisations.
- A feature engineered dataset incorporating additional environmental variables where appropriate.
- Machine learning classification models and at least one deep learning model for predicting food insecurity and energy insecurity.
- An explainability analysis identifying the key features influencing the model predictions.
- Interactive prediction web application
- A final report summarising:
  - Methodology
  - Results
  - Interpretation of findings
  - Key drivers of deprivation
  - Strengths and limitations of the developed models
  - Recommendations

# Notes

This project uses a real-world survey dataset that contains inconsistent survey responses and missing values, providing practical experience in real-world data cleaning.

External environmental datasets, such as crime statistics and green space measures, will be incorporated where appropriate using anonymised location information.

During feature engineering and model development, care will be taken to avoid data leakage. Collaborative development will be managed through Git and GitHub.
